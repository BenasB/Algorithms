# Algorithms

A simple collection of algorithms in C++ and C# that I wrote for 
practice.

## Features

### C#

* Graph
* Weighted Graph
* Priority Queue
* DFS (Recursive)
* DFS (Iterative)
* BFS
* Dijkstra
* Heap

### C++

*Works with c++11*
* Graph
* Weighted Graph
* DFS (Recursive)
* DFS (Iterative)
* BFS
* Dijkstra
* Flood Fill (Recursive)
* Flood Fill (Iterative)
